KB 1 · PENGERTIAN
🖥️

Apa itu
Hardware?

Hardware (perangkat keras) adalah semua jenis komponen pada sebuah perangkat komputer yang memiliki bentuk fisik — bisa dilihat, disentuh, dan dirasakan secara nyata.

💡 Hardware bekerja bersama software untuk menjalankan sistem komputer secara keseluruhan.

Tanpa hardware, program dan aplikasi tidak dapat berjalan. Sebaliknya, hardware tidak berguna tanpa software yang mengendalikannya.

KB 2 · PERANGKAT JARINGAN

Perangkat Keras Jaringan

📡

NIC (Network Interface Card)

Kartu jaringan yang terpasang di komputer untuk koneksi ke jaringan, tersedia versi wired (LAN) dan wireless (WiFi).

🌐

Router

Mengarahkan paket data antar jaringan. Berfungsi sebagai gateway antara jaringan lokal dan internet.

🔀

Switch

Menghubungkan banyak perangkat dalam LAN. Lebih cerdas dari hub karena mengirim data langsung ke tujuan.

📶

Access Point (WiFi)

Memperluas jangkauan jaringan nirkabel agar perangkat WiFi bisa terhubung ke jaringan yang ada.

🔌

Kabel Jaringan

Media transmisi fisik: kabel UTP (Cat5e/Cat6), kabel koaksial, dan fiber optik untuk kecepatan sangat tinggi.

📱

Modem

Mengubah sinyal digital ke analog dan sebaliknya, memungkinkan koneksi internet melalui ISP (provider).

KB 2 · TOPOLOGI JARINGAN

Jenis Topologi Jaringan

Bintang (Star)

Semua node terhubung ke satu pusat (switch/hub). Mudah dikelola, kerusakan satu node tidak memengaruhi lainnya.

Bus

Semua node terhubung ke satu kabel utama. Sederhana namun jika kabel pusat rusak, seluruh jaringan terganggu.

Cincin (Ring)

Setiap node terhubung ke dua node membentuk lingkaran. Data mengalir satu arah, performa stabil.

Jenis Jaringan:
LAN MAN WAN WLAN PAN
Local · Metropolitan · Wide · Wireless · Personal Area Network

KB 3 · SPESIFIKASI MINIMUM

Rekomendasi Spesifikasi Hardware

🖥️ Komputer Server

Prosesor Intel Xeon / Core i5 ke atas
RAM Minimal 8 GB (rekomendasi 16 GB)
Storage HDD 1 TB + SSD untuk sistem
OS Windows Server / Linux Ubuntu
UPS Terhubung langsung ke server

💻 Komputer Client/OPAC

Prosesor Core i3 atau setara
RAM Minimal 4 GB
Storage SSD 256 GB
Monitor 19 inci ke atas
OS Windows 10/11

🌐 Infrastruktur Jaringan

Kabel UTP Cat6 antar perangkat
Switch Minimal 8-port untuk workstation
Router Dilengkapi firewall keamanan
WiFi Access point untuk area baca
Bandwidth Minimal 20 Mbps dedicated

📦 Perangkat Pendukung

Scanner Barcode scanner USB/wireless
Printer Laser printer untuk label & laporan
Digitalisasi Flatbed scanner dokumen A4
Keamanan CCTV untuk pengawasan koleksi
Backup NAS / external drive untuk backup
